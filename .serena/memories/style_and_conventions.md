# Style and Conventions

- Rust 2024 edition idioms
- snake_case for functions/variables, CamelCase for types
- Commit messages: `type(scope): description` (feat, fix, docs, style, refactor, test, chore)
- No Co-Authored-By tags in commits
- Chat in Italian, develop and document in English
- CHANGELOG entries: `type(scope): description` grouped under date+version headers
